# 🛍️ E-Commerce API (Clean Architecture - ASP.NET Core)
This is a modular and scalable **E-Commerce REST API** built with **ASP.NET Core 8**, **Entity Framework Core**, and **Clean Architecture** principles.
---
## ✅ Features

- 🔐 JWT Authentication (planned)
- 🔍 Product Search, Filter, Sort, and Pagination
- 🛒 Shopping Cart and Checkout Process
- 💳 Stripe Payment Integration (planned)
- 📦 User Profiles with Order History
- 🗃️ SQL Server + EF Core Migrations
- 🧪 xUnit + Moq Unit Testing
- 🚀 Swagger for API Testing
- 🧱 Clean Architecture (Domain, Application, Infrastructure, API)
